Me too. I really like the waving wheat behind all the midwestern
states and the "don't fence me in" with the corn, wheat and
sunflowers which also shows the states. That was the whole idea to
start with - to create a Midwest Regional Patch.

Oklahoma is working on a state patch so I figured I'd offer two
cents worth of a Missouri patch as well.

Perhaps, I'll work on one of the Regional ideas that are in the
folder and send it off for a proof, as well, and see what folks
think of that.

It does seem to make a little bit of difference when they see it
embroidered nicely.

I will not be able to afford to buy a bunch of different styles of
patches without some commitment from people who might be interested
in purchasing them, because of the number of patches I would have to
buy of each one.

Still looking to the Great Lakes for inspiration...that's where it
all began for me...sigh...

Thanks,
~Perdu
