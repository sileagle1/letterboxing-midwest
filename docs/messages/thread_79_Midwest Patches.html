Yeah, it's still there, but you may need to join the group to see it.
Here's a link to try:

http://ph.groups.yahoo.com/group/LbPNW/photos/browse/ae9d

The nice thing about using the Yahoo groups are that all members can
contribute individually and you aren't dependent on someone else to
make the time to post the stuff. Anyone can add a file, link, photo
or poll whenever they want. On the Great Lakes group, we can also
easily create a database and any member can add a record. Not sure
why, but it doesn't look like you have the database feature enabled on
this group. We've used the database feature to list what campsites
people will be at for an event, the subject matter for stamps being
carved for an event (to avoid duplication), etc. It is a lot easier
to find this info if it is in a single place, rather than spread
across multiple email messages. I know the Midwest group is small
right now, but I just wanted to let you know about some of the nice
features on the Yahoo groups message boards.
